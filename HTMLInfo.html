    • Headings

    Heading tags are often used as the title of the pages (not to be misunderstood as the title tag, which is for adding a name to a webpage tab) and are meant to give the user and introduction to what they might see below the Heading.

    It's best practice to put Headings i n a size hierarchy, where the bigger Headings go above the smaller Headings, making it better for the user
